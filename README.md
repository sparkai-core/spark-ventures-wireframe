# Spark Ventures — Wireframe

Landing page wireframe + Next.js implementation for Spark Ventures.

## What's in here

- **`Spark Ventures Landing Page Wireframe.html`** — Updated wireframe (drag, drop, annotate)
- **`Product Page Wireframe.html`** — Original products page wireframe (reference)
- **`app/`** — Next.js 14 app router pages
- **`components/`** — React components (Marketplace, FAQ, etc.)
- **`lib/`** — Data sources (products, filters, etc.)

## Wireframe updates (latest)

- **Closing CTA** section replaced with the **Page Bottom CTA** pattern (`Seen something that fits?`)
- **Marketplace** section added under Portfolio — Case Studies (6 of 15 products, with filter pills and pagination bar)

## Local development

```bash
npm install
npm run dev
```

App starts on http://localhost:3000

## Tech

- Next.js 14.2.5
- React 18.3
- TypeScript
- Tailwind CSS 3.4
